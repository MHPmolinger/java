装饰 ： text-decoration
  • none：默认值
  • underline ：下划线
  • overline：上划线
  • line-through：删除线
display
  • none: 不显示，不占位置
  • block: 块元素显示
  • inline：内联元素显示
  • inline-block：行内块元素显示
visibility
  • visible ：默认值，在页面显示
  • hidden ：隐藏不显示，位置依然保持
overflow
  • visible：默认值，在父元素以外的位置显示
  • hidden： 溢出的内容，不会显示
  • scroll,：会为父元素添加滚动条
  • auto：根据需求自动添加滚动条

子元素选择器
p:first-child：选择第一个子元素p
p:last-child：选择最后一个子元素p
p:nth-child(odd)：选择所有奇数位置元素p
p:nth-child(even)：选择所有偶数位置元素p
p:first-of-type：选择每代第一个元素p
p:last-of-type：选择每代最后一个元素p
兄弟元素选择器
span + p：选择span后的第一个p元素
span ~ p：选择span后的所有p元素
选择器优先级
!important->内联样式->id选择器->类选择器->通配选择器

伪元素
  • p:first-letter： 为p中第一个字符来设置一个样式
  • p:first-line ： 为p中的第一行设置一个样式
  • p:before： 元素最前边加入内容
    • p:before{ content: "我会出现在整个段落的最前边"; color: red; }
  • p:after ： 元素最后边加入内容
    • p:after{ content: "我会出现在整个段落的最后边"; color: red; }
