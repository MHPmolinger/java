语言分类
DQL：Date Query Language
包括：select
作用：从表中检索数据
DML：Date Manipulation Language
包括：insert、update、delete
作用：添加，修改，删除表中的数据
TPL：Transaction Process Language
包括：commit、rollback
作用：提交，回滚
DCL：Date Control Language
包括：grant、revoke
作用：授权，收回权限
DDL：Date Definition Language
包括：create、drop、alter、truncate
作用：定义，删除，修改，截取数据库对象

  4. is null与is not null

  6. like

        3. 查询条件含有单行函数
